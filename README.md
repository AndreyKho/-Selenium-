# Selenium Python Automation Course 🚀

Репозиторий содержит решения практических заданий курса **"Автоматизация тестирования с помощью Selenium и Python"**.

## 📑 Содержание

**Основные методы Selenium**  
Работа с кнопками, чекбоксами и радиобаттонами

**Ожидания в Selenium**  
Использование Implicit Waits и Explicit Waits (WebDriverWait)

**Работа с окнами**  
Переключение между вкладками и обработка алертов

**Финальные проекты**  
Комплексные тестовые сценарии

## 🛠 Технологии в проекте

| Инструмент | Описание |
|------------|----------|
| Python 3.x | Основной язык программирования |
| Selenium WebDriver | Инструмент для автоматизации действий в браузере |
| PyTest | Фреймворк для запуска и организации тестов |
