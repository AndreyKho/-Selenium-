Selenium Python Automation Course 🚀
Репозиторий содержит решения практических заданий курса "Автоматизация тестирования с помощью Selenium и Python".

📑 Содержание
Основные методы Selenium: работа с кнопками, чекбоксами и радиобаттонами.

Ожидания в Selenium: использование Implicit Waits и Explicit Waits (WebDriverWait).

Работа с окнами: переключение между вкладками и обработка алертов.

Финальные проекты: комплексные тестовые сценарии.

🛠 Технологии в проекте
Python 3.x — основной язык программирования.

Selenium WebDriver — инструмент для автоматизации действий в браузере.

PyTest — фреймворк для запуска и организации тестов.

🚀 Как запустить тесты локально
Клонируйте репозиторий:

Bash
git clone https://github.com/твой_логин/название_репозитория.git
cd название_репозитория
Создайте и активируйте виртуальное окружение:

Bash
python -m venv .venv
source .venv/bin/activate  # для macOS/Linux
# или
.venv\Scripts\activate     # для Windows
Установите зависимости:

Bash
pip install selenium pytest
Запустите нужный скрипт:

Bash
python "Основные методы Selenium/lesson2.4/task1.py"
